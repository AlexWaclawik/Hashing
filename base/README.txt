HOW TO RUN PROGRAM:
	* Type "make" then "make run"
	* input a single word (case sensitive)
	* Program will then look for the word in the dictionary
	* If it exists, the program will suggest similiar words that start with the same two letters


MAKEFILE COMMANDS:
	* "make" - compiles program
	* "make run" - runs program
	* "make clean" - cleans up previous compile
